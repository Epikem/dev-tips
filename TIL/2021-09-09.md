다만, 높이를 N을 곱해야 하나 아니면 N-cur를 해야 하나?

```py

def solve(food_times, k):
    cur = -1
    left = 0
    N = len(food_times)
    cols = []
    for i,v in enumerate(food_times):
        left+=v
        cols.append((v, i+1))

    cols.sort()
    diffs = [cols[0][0]]
    for i in range(N-1):
        diffs.append(cols[i+1][0] - cols[i][0])
    cur = 0
    while True:
        if(cur == N):
            break
        nextv = diffs[cur]

        if(k>=nextv * (N-cur)):
            k -= nextv * (N-cur)
            cur+=1
        else:
            break
        pass

    cols = cols[cur:]
    
    cols.sort(key=takeSecond)

    if(N==cur):
        return -1
    x = math.floor(k/(N-cur))

    k -= x*(N-cur)
    
    return cols[k][1]
```
