#### 디버깅

```py
a = list(range(1, 100000))
a=a+[10001] # O(N)
a.append(10001) # O(1)

a=a+[1,2,3] # O(N+k)
a.extend([1,2,3] # O(k))

a += [4,5,6] # O(N+k)? -> 알아서 extend를 실행하므로 O(k)

if 10 in a: # O(N)
    print(1)

print(len(a)) # O(1)

```
